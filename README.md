## About The Project

![Portfolio Screenshot](./public/projects/connectly.png)

This project represents a shift from "building components" to **"architecting systems."** It serves as a living proof-of-concept for the **2026 Web Engineering Gold Standard**, utilizing the latest bleeding-edge technologies to deliver an application that is performant, scalable, and resilient.

Unlike traditional portfolios, this application adopts **Feature-Sliced Design (FSD)** to decouple business logic from UI, ensuring that the codebase remains maintainable as it scales. It leverages a hybrid animation model (GSAP + Framer Motion) for optical stability and fully embraces the **React 19** ecosystem.
